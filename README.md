# Generative AI Portfolio

A collection of projects demonstrating skills in modern generative AI and deep learning techniques.

## Projects

### 1. [PyTorch & GPT-2 Fundamentals](PyTorch_GPT2_Fundamentals.ipynb)
Foundational skills in PyTorch and language model manipulation using OpenAI's GPT-2.
- PyTorch tensor operations and neural network construction
- Building classification networks with multiple hidden layers
- GPT-2 tokenization and vocabulary handling
- Next-word prediction and probability calculations
- Cross-entropy loss computation and softmax temperature scaling

### 2. [LLM Pretraining & Fine-Tuning](LLM_Pretraining_FineTuning.ipynb)
Pretraining and fine-tuning techniques for large language models.
- Continued pretraining of GPT-2 on domain-specific data (movie plots)
- Measuring perplexity changes across domains
- Fine-tuning for sequence classification tasks
- Sentiment analysis with custom classifiers
- Training loop implementation and evaluation

### 3. [QLoRA: Parameter-Efficient Fine-Tuning](QLoRA_Parameter_Efficient_FineTuning.ipynb)
Parameter-efficient fine-tuning (PEFT) using QLoRA (Quantized Low-Rank Adaptation).
- Low-Rank Adaptation (LoRA) implementation
- Model quantization techniques (4-bit, 8-bit)
- Fine-tuning GPT-2 Large with minimal parameters
- Memory-efficient training strategies
- Sentiment classification with PEFT

### 4. [Multimodal AI: Image Generation & CLIP](Multimodal_Image_Generation_CLIP.ipynb)
Multimodal AI capabilities including text-to-image generation and image understanding.
- Text-to-image generation with diffusion models
- CLIP for zero-shot image classification
- Visual Question Answering (VQA) implementation
- Automated image evaluation pipelines
- Prompt engineering for image generation

### 5. [RAG System Implementation](RAG_System_Implementation.ipynb)
Complete Retrieval-Augmented Generation (RAG) system for enterprise question-answering.
- RAG architecture design and implementation
- Vector database integration for document retrieval
- LangChain/LangGraph pipeline construction
- Embedding models and similarity search
- LLM integration for answer generation
- RAG evaluation metrics

## Technologies Used
- **Deep Learning**: PyTorch, Hugging Face Transformers
- **LLMs**: GPT-2, Mistral, Qwen
- **PEFT**: LoRA, QLoRA, BitsAndBytes
- **Multimodal**: CLIP, Stable Diffusion, VQA models
- **RAG**: LangChain, Vector Databases, Embedding Models
